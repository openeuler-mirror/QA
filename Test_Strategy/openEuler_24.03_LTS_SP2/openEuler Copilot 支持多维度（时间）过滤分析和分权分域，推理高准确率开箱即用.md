# 特性描述
<!-- 主要介绍特性实现的背景、功能以及作用 -->

EulerCopilot 支持企业拖拉拽方式构建企业知识问答。

功能：

1、建设数据治理流程，数据集评估流水线

2、MCP server 框架集成

3、支持应用配置多知识库，知识库可以基于权限管理，实现多用户空间管；

4、Copilot向量数据库支持openGauss
