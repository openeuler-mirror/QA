# 特性描述
Raspberry Pi（树莓派）是由 Raspberry Pi 基金会与 Broadcom 公司联合开发的一系列小型单板计算机，凭借价格低廉、体积小巧、功耗低、可编程性强以及生态系统丰富等优势，已广泛应用于工业自动化、机器人技术、物联网、教育及创客项目等领域。openEuler 官方支持当前主流的树莓派型号——树莓派 4B 和树莓派 5，充分发挥其 ARM 架构、低功耗、高性价比及丰富外设生态的优势。本测试策略旨在全面验证 openEuler 在树莓派平台上的功能完整性和硬件兼容性，加速其在 ARM 架构边缘计算设备生态中的落地与推广。

# 特性分层策略
## 总体测试策略
- 测试目标：本次测试主要覆盖功能测试和硬件兼容性测试。功能测试包括：镜像安装，系统基本信息查看，用户功能，软件管理功能，服务管理功能，进程管理功能，网络管理功能，开发环境等；硬件兼容性测试包括：树莓派 4B/5 开发板的 USB 接口、HDMI 接口、以太网接口、Wi-Fi 、蓝牙的兼容性验证。
- 时间进度：待评估
- 风险评估：NA
- 重点事项：同测试目标
- 争议事项：NA

## 功能测试
| 功能描述 | 设计思路 | 测试重点 | 备注 |
| ------- | ------- | ------- | ---- |
| 系统安装 | 镜像刷写到树莓派 SD 卡并引导启动 | 能正常刷写镜像；设备可成功上电启动；控制台或图形界面可正常登入；内核与硬件匹配无报错 |  |
| 基本功能及配置 | 验证系统基础运行能力与用户交互配置功能 | 用户登录（本地/SSH）正常；时区、主机名、语言等基础配置可修改并生效；文件系统读写正常；关机/重启指令有效 | |
| 软件管理 | 使用 DNF/YUM 包管理器进行软件安装、更新与卸载 | 能正常联网访问 openEuler 软件源；dnf install/update/remove 命令执行成功；依赖解析正确；已安装软件可正常运行 | |
| 服务管理 | 通过 systemd 管理系统服务的启停与状态监控 | systemctl start/stop/status/restart 命令对关键服务（如 sshd、firewalld）有效；服务开机自启配置可持久化 | |
| 进程管理 | 对系统进程进行查看、调度与终止操作 | ps、top、htop、kill、pkill 等命令可用；能正确识别和管理系统进程；资源占用显示准确 | |
| 网络管理 | 配置有线/无线网络连接及网络服务访问 | 以太网自动获取 IP 正常；Wi-Fi 可连接并联网；ping、curl、ssh 等网络工具可用；防火墙规则可配置 | |
| 开发环境 | 提供基础开发工具链与运行时支持 | 预装或可安装 gcc、make、gdb、git、python3 等开发工具；能编译运行 C/C++/Python 示例程序 | |

## 硬件兼容性测试
| 测试目标 | 设计思路 | 测试重点 | 备注 |
| ------- | ------- | ------- | ---- |
| USB 接口兼容性 | 验证 USB 2.0/3.0 接口对常见外设的识别与数据传输能力 | 能正常识别并挂载 U 盘、移动硬盘、USB 键鼠、摄像头等设备；文件读写稳定；多设备同时接入无冲突 | |
| HDMI 视频输出 | 验证 HDMI 接口可用性 | 视频功能正常 | |
| 以太网接口 | 验证有线连接可用性 | 网卡自动识别并获取 IP（DHCP）；支持静态 IP 配置 | |
| Wi-Fi 功能 | 验证板载 Wi-Fi 模块的连接与通信功能 | 能扫描并连接无线网络；联网访问正常；信号强度显示准确 | |
| 蓝牙功能 | 验证板载 Bluetooth 模块的可用性 | 能启用蓝牙服务（bluetoothd）；可以使用 bluetoothctl 扫描、配对并连接蓝牙设备 | |
