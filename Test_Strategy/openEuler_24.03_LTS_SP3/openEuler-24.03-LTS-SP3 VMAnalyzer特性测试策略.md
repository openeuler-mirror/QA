# 特性描述

VMAnalyzer是由移动云操作系统团队研发的一款轻量级的虚拟化性能监测分析工具，能够细粒度的分析虚拟机的运行状况和性能

【特性竞争力】
可轻松识别性能问题和瓶颈，可以帮助用户维护具有高性能和高可靠性的虚拟机

## 特性应用场景分析
1. 虚拟化环境中虚拟机运行状况和性能采集检测
2. 宿主机信息/虚拟化环境信息/虚机基本信息采集检测

## 功能测试
| 功能描述 | 设计思路 | 测试重点 | 备注 |
| ------- | ------- | ------- | ---- |
| 虚拟机性能数据采集 | 测试VMAnalyzer提供的虚拟机性能数据采集功能 | 能够正常轮询采集并输出虚拟机内存、CPU、磁盘、网络等信息  |      |
| 虚拟机基本信息采集 | 测试VMAnalyzer提供的虚拟机基本信息采集功能| 能够正常采集并输出虚拟机绑核、iothread、fsinfo等信息       |      |
