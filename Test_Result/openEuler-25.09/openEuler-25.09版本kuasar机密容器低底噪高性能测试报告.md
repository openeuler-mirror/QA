# 3     测试结论概述

## 3.1   测试整体结论

kuasar机密容器低底噪，高性能特性，共计执行两项测试，主要覆盖了管里面内存底噪测试和启动时间测试。kuasar机密容器相比kata机密容器，降低容器管理面底噪50%；kuasar机密容器相比kata机密容器，提升启动性能21.43%。

| 测试活动 | 测试子项 | 活动评价 |
| ------- | -------- | ------- |
| 性能测试 |   拉起两个sandbox，查看管理面底噪是否符合预期--降低50%      |  测试通过       |
| 性能测试 |   启动一个机密容器，查看启动性能是否符合预期--提升20%      |  测试通过       |

## 3.2   约束说明

需要物理环境，需要支持TEE。
