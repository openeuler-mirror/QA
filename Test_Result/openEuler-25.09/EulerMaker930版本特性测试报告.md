# 1     特性概述

EulerMaker精简repo相关组件，合并new-repo和publisher到repo-manager中；下线redis组件，使用已有etcd组件接管redis的能力；解耦服务启动依赖，实现自恢复能力；残留任务队列清理；解决etcd偶现卡死问题。
