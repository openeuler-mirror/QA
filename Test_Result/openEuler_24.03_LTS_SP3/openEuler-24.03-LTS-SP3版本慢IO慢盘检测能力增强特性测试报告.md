关键词： sysSentry

摘要：sysSentry主要提供故障巡检框架，该框架通过提供统一的北向故障上报接口以及南向提供支持不同巡检/诊断能力的插件，支持对系统中CPU、内存、磁盘、NPU等硬件故障进行巡检和诊断。


缩略语清单：

| 缩略语 | 英文全名 | 中文解释 |
| ------ | -------- | -------- |
|   sysSentry      |    sysSentry      | 系统级故障管理框架         |
|        |          |          |

# 1     特性概述

本特性优化慢IO告警日志输出格式，将开发者日志和用户日志区分，降低用户理解门槛;支持硬盘单IO故障检测，当前出现单个IO时延极端异常情况能及时发现并上报告警;针对特定型号的硬盘，产生latency告警事件时，在用户日志中增加磁盘侧的IO时延分布统计信息。
