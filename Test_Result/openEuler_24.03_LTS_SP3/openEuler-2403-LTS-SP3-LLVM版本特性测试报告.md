# 1     特性概述

openEuler 2403 LTS SP3版本新增转测3个特性, 其中每个特性下包含两个子特性, 下面对每个特性做简要介绍:

- **LLVM编译器易用性增强**: openEuler 24.03 LTS SP3**LLVM编译器易用性增强**特性包含以下两个特性:
  - **clang tidy 工具告警增强**: 为了提高客户开发迁移效率，需要持续增强LLVM对GNU编译器的兼容性，减少迁移成本, 添加clang tidy工具对7类典型场景进行分析并给出修改建议。
  - **PGO 反馈优化可用性增强**: 为提高PGO易用性，降低使用者的使用难度，减少使用流程步骤, 新增可用性增强特性, 针对两类典型场景, 进行PGO采样流程标准化。
- **LLVM编译器提升数据库(MySQL)应用性能2-5%**: openEuler 24.03 LTS SP3 **LLVM编译器提升数据库(MySQL)应用性能2-5%**特性包含以下两个特性:
  - **REV16指令合并优化**: 针对在小端和大端字节序之间切换时常见的一些模式，进行指令层面的优化合并, 以减少生成的指令数量, 以提高吞吐量，并减少在小端和大端字节序切换时生成的加载和存储指令的数量。
  - **BOLT AArch64优化能力增强**: 回合上游LLVM社区BOLT相关优化补丁以及bugfix。
- **LLVM编译器提升压缩解压缩lz4应用性能5%**: openEuler 24.03 LTS SP3 **LLVM编译器提升压缩解压缩lz4应用性能5%**特性包含以下两个特性:
  - **AArch64 loop idiom优化, 提升xz应用性能3-8%**: 回合loop idiom优化特性, 优化xz应用性能。
  - **machine sink优化&BBplacement优化增强, 提升LZ4应用特性性能3-8%**: 本特性包括提供提供machine sink下沉操作的禁用选项, 减少或避免将热点循环中读取后相关指针移动的操作下沉到循环末尾造成的性能下降问题, 并对rotate loop 进行了增强, 以提升BlockPlacement 优化的效果
