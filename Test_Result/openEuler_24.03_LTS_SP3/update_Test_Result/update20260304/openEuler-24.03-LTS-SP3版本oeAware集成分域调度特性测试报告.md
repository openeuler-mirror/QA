# 1     特性概述

分域调度特性通过 将进程/容器的任务调度限制在用户指定大小的 CPU 组（软域），减少跨节点的缓存同步与内存访问开销，从而提升 高并发、低延迟业务（如数据库、Web 服务、实时计算）的性能与确定性。oeAware 集成分域调度特性的正确性、稳定性、安全性和推荐能力，确保其在实际生产环境中可靠运行。

# 3     测试结论概述

## 3.1   测试整体结论

分域调度（soft_domain_tune）特性共计执行12 个测试用例，主要覆盖了 ​功能验证、配置边界、异常容错、平台兼容性及稳定性测试。经过测试​，发现 1个缺陷问题（对运行过程中创建的线程不能动态修改）均已修复并回归通过。当前版本在 Kunpeng 920 平台上运行稳定，无遗留高/中风险问题，整体质量良好，满足 openEuler 24.03 LTS SP3 发布要求。
