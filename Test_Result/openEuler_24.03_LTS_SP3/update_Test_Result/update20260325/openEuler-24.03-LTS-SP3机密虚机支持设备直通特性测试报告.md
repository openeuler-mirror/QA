 	 关键词： CCA套件
 	 
 	 摘要：机密计算是一种基于可信硬件，保护数据处理安全的一种技术，对应的ARM架构处理器应用的是TrustZone技术。
 	 传统TrustZone可信执行环境仅面向终端设备，TA应用必须经过终端设备厂商认证后才可以在TEE（Trusted Execution Environment，可信执行环境）OS侧运行，不支持通用计算场景。CCA套件则是基于虚拟机的下一代机密计算技术，能够兼容已有的应用生态，并将机密计算的载体从应用层扩展到操作系统层。

 	 # 1     特性概述
 	 
 	 鲲鹏服务器面向互联网和国计民生业务场景，需要构建数据安全防护能力，打造安全算力底座，依赖机密计算能力。鲲鹏950服务器采用了ARMv9硬件架构，可以支持使能CCA机密计算技术，以实现安全算力底座

 	 # 3     测试结论概述
 	 
 	 ## 3.1   测试整体结论
 	 
 	 CCA机密虚机支持设备直通特性，共计执行11个用例，主要覆盖了功能测试和场景测试和资料测试，测试过程中未发现问题，测试通过，无遗留风险，整体质量良好
 	 
 	 ## 3.2   约束说明
 	 | 限制 | 约束说明 | 备注 |
 	 | -------- | ------------ | ---- |
 	 |    应用限制      |   硬件服务器需搭载鲲鹏950处理器           |      |
 	 |    不支持超线程      |   当用户使能CPU超线程能力后，可能存在针对机密计算的侧信道攻击等风险，因此业界建议在禁用SMT的系统上运行机密计算           |      |
 	 |    不支持超分配      |   CPU和内存超分配当前不支持           |      |
