# 1     特性概述

UBS uCache针对裸机容器场景，基于灵衢总线构建全局PageCache池，I/O敏感应用节点通过灵衢总线借用邻居节点扩充可用PageCache空间，并且在Linux内核PageCache顺序预取的基础上，优化不同文件顺序读场景的性能，提升存算一体型应用的IO效率。
