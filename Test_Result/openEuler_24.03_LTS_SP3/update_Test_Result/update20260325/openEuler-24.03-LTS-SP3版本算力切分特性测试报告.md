# 1     特性概述

算力切分特性实现AI任务包括Model、Stream、Kernel等抽象，任务到异构算力按需调度机制，构建包括异构抢占，时间片切分，组调度，带宽管控等调度机制，并实现异构算力带宽管控，对接云原生容器生态。

## 3.2   约束说明

1. 仅支持小模型混部场景，即多个小模型使用同一张NPU卡；
2. 仅支持NPU卡，且不支持设备动态插拔场景，各个型号NPU卡需要进行适配；
3. 提供RT和CFS调度算法，只支持在算子边界抢占，不支持硬件抢占。
4. 禁止在未使能xcu的cgroup组内添加需要使用算力切分特性的任务，否则会出现非预期的行为。
5. 仅一级cgroup支持修改调度类，其内部的子cgroup默认继承一级的xcu.sched_class，当该一级cgroup内存在子cgroup或普通任务时，无法修改调度类。
