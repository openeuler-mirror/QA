# 1     特性概述

ub-pkg-manager 是一套用于管理、部署和监控 UB OS 软件包的工具集，包含多个功能模块和一个统一的命令行界面。实现UB组件的快速安装部署与驱动自动加载。提供ub-pkg-cli命令用于配置UB相关组件的配置参数与基础功能检测能力。[openEuler源码仓地址](https://atomgit.com/openeuler/ub-pkg-manager)（README）
