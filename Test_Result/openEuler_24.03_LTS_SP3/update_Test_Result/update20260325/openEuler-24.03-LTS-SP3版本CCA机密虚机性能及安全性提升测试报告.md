关键词： 
CCA、机密计算

摘要：
支持CCA机密计算能力，基于大页内存映射优化CCA机密虚机启动及运行性能，提供虚机度量启动及远程证明机制

|  CCA   |   Confidential Compute Architecture       |    机密计算架构     |

# 1     特性概述

基于大页内存映射优化CCA机密虚机启动及运行性能满足性能要求，提供CCA机密虚机度量启动及远程证明机制，能够证明机密虚机和机密计算平台的可信程度

## 3.1   测试整体结论

## 3.2   约束说明

| 限制 | 约束说明 | 备注 |
| -------- | ------------ | ---- |
|    应用限制      |   硬件服务器需搭载鲲鹏950处理器           |      |
|    不支持超线程      |   当用户使能CPU超线程能力后，可能存在针对机密计算的侧信道攻击等风险，因此业界建议在禁用SMT的系统上运行机密计算           |      |
