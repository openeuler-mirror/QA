# 1     特性概述

在字节 Pack 微服务场景中，鲲鹏芯片（920/950等）性能与友商产品（AMD 9Y24等）存在一定性能差距，影响客户后续采购意愿，需要针对 Golang 编译器进行优化，实现性能反超友商，重获优势。

根据客户实际业务热点采样数据分析，其热点在 HashMap 上约占 10%~20%，存在优化价值，开发团队应当从向量化优化、鲲鹏亲和指令优化、数据结构调整优化等维度展开优化特性开发，并以开源或自研 bench 作为评估性能提升的依据。
