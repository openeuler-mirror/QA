# 1     特性概述

CVE AI智能修复特性是openEuler社区sig-intelligence团队开发的CVE漏洞智能修复工具，基于MCP（Model Context Protocol）协议实现。该工具能够自动化分析CVE漏洞信息，从上游社区获取补丁，并支持将补丁回移植到目标分支，主要功能包括：

- CVE信息自动获取与分析
- 补丁自动获取与解析
- 补丁回移植（backport）功能
- 分支管理与代码同步
- 补丁应用与验证
