# 1     特性概述

mcp-server自动辅助生成特性是openEuler社区sig-Intelligence团队开发的MCP Servers自动化生成工具，旨在帮助开发者快速创建符合MCP协议的服务器。该特性主要包含两种生成模式，能够显著降低MCP Servers的开发门槛，提高开发效率。

## 1.1 生成模式说明

### 基于软件包使用文档的生成（manual_mcp_generator.py）

该模式通过解析软件包的使用文档（man手册或帮助文档），自动提取常用命令和参数信息，生成对应的MCP Servers代码。主要功能包括：
- 软件包使用文档自动获取与解析
- 常用命令和参数的智能提取
- MCP Servers代码框架自动生成
- 生成代码的语法校验与自动重试机制

### 基于测试套件的生成（testsuite_mcp_generator.py）

该模式通过分析社区软件包的测试套件测试用例，自动生成用于测试的MCP Servers。主要功能包括：
- 测试套件数据自动获取与收集
- 测试用例结构分析与提取
- MCP Servers代码框架自动生成
- 生成代码的语法校验与自动重试机制

## 1.2 核心能力

- **代码自动生成**：借助大模型能力，自动生成符合MCP协议的服务器代码及相关配置文件
- **一键生成脚本**：提供简洁易用的一键生成脚本，降低用户操作复杂度
- **测试框架集成**：支持与社区MCP测试框架mcp-testkit无缝集成，实现自动化生成和测试
- **多源输入支持**：支持从软件包文档和测试套件两种来源自动提取信息生成MCP Servers

1. 特性需保证在openEuler社区范围内依赖自闭环；反之需在TC、RM获取例外结论后，提供完整的二进制依赖，保证所有依赖公开可获得。
