# 1     特性概述

DevStation 24.03 SP3 DMG构建用于将 qcow2 虚拟机镜像打包为 macOS 可安装的 DMG 文件。主要功能包括：

1. **UTM 配置生成**：基于 qcow2 文件自动生成 UTM 虚拟机配置文件（config.plist）和文件夹结构。
2. **DMG 打包**：将生成的 UTM 文件夹、安装脚本和资源文件打包为 DMG 磁盘映像，方便用户拖拽安装。
3. **自定义选项**：支持自定义输出文件名、额外文件目录、应用程序图标等。
4. **错误处理**：对缺失文件、无效参数等异常情况进行友好提示。

## 3.2   约束说明

1. **操作系统**：仅支持 macOS（依赖 create-dmg 和 hdiutil两个mac工具）。
2. **依赖工具**：需要提前安装 create-dmg（可通过 Homebrew 安装）。
3. **输入文件**：iso或qcow2 文件。
4. **磁盘空间**：构建过程中需要额外空间存放临时文件和最终 DMG，不小于20G。
