## 3.2   约束说明

| 约束项 | 约束说明 | 影响范围 |
| ------ | -------- | -------- |
| CPU 架构 | 仅支持 ARM64 架构 | 非 ARM64 平台无法使用 |
| CPU 型号 | 仅支持 鲲鹏920高性能版 | 其他 CPU 型号加载模块会失败 |
| 内核版本 | 支持 5.10 内核 (openEuler 22.03) 和 6.6 内核 (openEuler 24.03) | 驱动需根据源码仓编译 |
| 权限要求 | 需要 root 权限操作 sysfs 接口 | 普通用户只能读取状态 |
| 硬件依赖 | 需要 Kunpeng 920 7270Z或其他鲲鹏920高性能版机器 | 其他硬件可能寄存器地址不匹配 |
