关键词： RISC-V, LZ4, Zicclsm, 内存访问优化, 性能优化

摘要：
本文档详细记录了针对支持Zicclsm扩展的RISC-V处理器优化LZ4压缩库内存访问方式的特性测试结果。实际测试表明，在openEuler RISC-V平台上，通过启用直接内存访问模式，LZ4压缩性能相比未优化版本提升31.8%，达到了预期优化目标，同时保持了数据完整性和解压性能稳定。


缩略语清单：

| 缩略语 | 英文全名 | 中文解释 |
| ------ | -------- | -------- |
| LZ4 | Lempel-Ziv 4 | 快速无损压缩算法 |
| RISC-V  |  Reduced Instruction Set Computer - V  |  精简指令集计算机第五代 |
| Zicclsm  | RISC-V Misaligned Atomics Extension | RISC-V未对齐访问扩展 |

# 1     特性概述

本特性针对支持Zicclsm扩展的RISC-V处理器，优化了LZ4压缩库的内存访问方式。通过利用RISC-V架构的Zicclsm扩展（未对齐原子操作扩展），实现了对未对齐内存访问的硬件加速支持。特性修改主要包括在LZ4代码中使用直接未对齐内存访问作为RISC-V平台上的最优方法，类似于ARMv6架构的实现方式。优化后的代码在保持API兼容性的同时，显著提升了压缩性能。

## 3.2   约束说明

- 该特性依赖于RISC-V处理器的Zicclsm扩展支持
- 需要适当的编译器支持以识别Zicclsm扩展
- 在不支持Zicclsm的RISC-V处理器上，该特性会自动回退到标准内存访问模式，不会导致功能异常
