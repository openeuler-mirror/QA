# 1     特性概述

openEuler 24.03 LTS SP3版本Heolleo是一个现代化的操作系统安装客户端，旨在为开发者提供便捷、高效的devstation开发环境安装体验。采用Electron + Vue 3 + TypeScript技术栈，提供现代化用户界面、硬件识别分析、跨平台架构以及高性能安装引擎。

## 1.1 特性

#### 现代化用户界面
- 基于Vue 3和Element Plus构建的响应式UI
- 支持中英文国际化（i18n）
- 图形化分区管理、硬件信息展示
- 安装进度可视化

#### 智能安装与配置
- 基于硬件指纹的自动配置推荐
- 基本安装和桌面环境安装
- 自动化分区方案（手动/自动）
- 用户账户与密码管理

#### 高性能安装引擎
- 底层使用Python编写的eulerinstall子项目
- 并行文件解压与zstd压缩算法加速
- 支持多种文件系统（ext4, btrfs, fat32等）
- 网络镜像自动选择与优化

#### 跨平台与兼容性
- 支持x86_64、AArch64架构
- 适配openEuler 24.03 LTS SP3及衍生版本
- 与主流硬件厂商服务器兼容

#### 安全与可靠性
- 密码加密存储与传输
- 硬件特性自检
- 日志记录与错误诊断

## 3.2   约束说明

- Heolleo依赖Electron运行时，需预装Node.js环境（仅开发构建阶段）
