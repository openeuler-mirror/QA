# 3     测试结论概述

## 3.1   测试整体结论

kernel-ci特性，保障openEuler及上游内核构建、启动和功能验性，共计执行20个用例，主要覆盖了手动触发与webhook触发kernel构建的功能测试和性能测试，共发现问题6个，回归通过问题5个，遗留问题1个，整体质量良好。
