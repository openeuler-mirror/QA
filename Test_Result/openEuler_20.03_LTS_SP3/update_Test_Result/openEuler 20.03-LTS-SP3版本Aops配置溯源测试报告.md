# 1     特性概述

Aops配置溯源特性主要支持在大批量管理节点配置文件的场景下，提供文件模板基线化功能，方便客户对配置文件进行统一管理，对配置文件的异常修改能提供审计信息，且支持基线化配置文件通过SSH可以同步到节点管理主机。对于文件配置错误类引发的故障能够起到很好的支撑作用。
