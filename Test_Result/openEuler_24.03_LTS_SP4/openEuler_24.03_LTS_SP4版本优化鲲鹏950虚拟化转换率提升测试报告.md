## 5.2   测试流程

### 5.2.1 测试场景

测试规格：虚机规格 4U8G，分别测试单虚机和多虚机（96VM）场景。

### 5.2.2 计算性能测试

使用SPECint2017基准测试工具，分别在Kunpeng 950和AMD 9654环境下测试：
- 单VM计算性能
- 整机计算性能

### 5.2.3 内存性能测试

使用STREAM工具测试内存带宽：
- 赋值（Copy）
- 混合运算（Triad）

### 5.2.4 网络性能测试

使用Iperf工具测试网络性能：
- UDP时延测试（us）
- 包率测试（64B, pps）

### 5.2.5 存储性能测试

使用Fio工具测试存储性能：
- 4K随机写/读（IOPS）
- 4K随机写/读时延（us）````

### 5.2.6 线性度性能测试

使用Unixbench工具测试线性度

### 5.2.7 综合转换率计算

综合转换率计算公式：
```
综合转换率 = SQRT(线性度 × 综合虚拟化损耗)
综合虚拟化损耗 = AVERAGE(计算损耗 + 赋值损耗 + 混合运算损耗 + 包率损耗 + UDP时延损耗 + 4K随机读写损耗 + 4K随机读写时延损耗)
```

性能比计算：Kunpeng 950 / AMD 9654(2P) = 优化后综合转换率 / AMD综合转换率

测试结果：基线性能比0.87，目标1.03，优化后实际达到1.04，达成目标。
