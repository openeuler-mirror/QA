# 1     特性概述

在KP950 + velinux + Redis 7.0.15 + kraio优化后，redis-bankmark通过网卡IP对本地Redis 7.0.15单实例压测get、set，Redis 线程上仍有%5−20%的epoll_wait热点开销，通过将epoll_wait上监听的事件转移到kraio的Poll上，减消epoll_wait热点的开销，Redis本地网络吞吐性能提升5% 。
