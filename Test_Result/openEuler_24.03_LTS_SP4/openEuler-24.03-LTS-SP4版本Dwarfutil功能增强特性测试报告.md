# 1     特性概述

用户使用llvm编译器编译构建业务应用时，反映普遍存在编译时间过长的问题，期望能够提升编译速度。并行编译技术虽然能够显著提升编译速度，但是会使得Debuginfo size 膨胀，导致链接时崩溃。llvm-dwarfutil 是 LLVM 工具链中用于处理和优化调试信息（Debug Info）的核心工具。本特性针对llvm-dwarfutil 工具进行了增强，支持处理 Debug Types。该增强功能能在完整保留调试能力的前提下，精准识别并消除冗余的调试条目，有效减小debuginfo size。

## 3.2   约束说明

1. 此特性的目标为支持llvm-dwarfutil处理和生成type unit，其他未受支持的DWARF section不做修改；
2. 其他DWARF5新增的DWARF section并未被llvm-dwarfutil支持，本特性暂未支持对DWARF5中type unit的支持。
