# 1     特性概述

针对DPU场景的管理面进程卸载场景，需要将管理面进程迁移卸载到DPU上，而业务进程依然在HOST端运行，
为HOST释放部分CPU和内存资源，配合网络和存储的卸载，架构上实现管理和业务的彻底分离，降低管理面TCO，减少对业务的干扰
